# Halcones Estratégicos - Hexaquest: El Desafío del Vacío

## Descripción del minijuego
**Hexaquest: El Desafío del Vacío** es un minijuego de plataformas y reflejos desarrollado en Unity, en el que el jugador debe atravesar una plataforma mágica compuesta por hexágonos flotantes sin caer al vacío.

La experiencia se centra en el movimiento, el equilibrio, la rapidez de reacción y la toma de decisiones bajo presión. A medida que avanza, el jugador debe identificar rutas seguras, reaccionar ante cambios del entorno y evitar plataformas inestables o que desaparecen.

## Objetivo del juego
El objetivo principal es atravesar la plataforma de hexágonos flotantes sin perder todas las vidas. El jugador debe moverse, saltar y reaccionar correctamente para evitar caer al vacío y completar el recorrido.

## Mecánicas principales
- Movimiento del personaje
- Salto entre plataformas
- Sistema de 3 vidas
- Reaparición del jugador
- Plataformas hexagonales inestables
- Cámara isométrica
- Retroalimentación visual y sonora

## Elementos implementados
- Escenario de plataformas hexagonales
- Personaje principal
- HUD de vidas
- Materiales y ambientación visual
- Recursos de audio y efectos del entorno
- Integración del proyecto mediante ramas en GitHub

## Controles básicos
- **W / A / S / D** o flechas: mover al personaje
- **Space**: saltar

> Nota: los controles pueden variar según la configuración final del proyecto.
